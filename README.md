# Hamming_coding

Это консольное приложение позволяет декодировать код Хэмминга. Принимает на вход бинарное валидное сообщение классического кода Хэмминга (длина сообщения может быть 3, 7, 15 и т.д.).

## Установка и запуск

Для установки и запуска приложения требуется наличие Java Development Kit (JDK) версии 17 или выше.

Для сборки приложения достаточно запустить команду `mvn -B package --file pom.xml`. После успешной сборки приложения можно найти .jar файл в директории `target`.

Для запуска приложения необходимо выполнить команду `java -jar <имя jar-файла>.jar <входное сообщение>`, где `<входное сообщение>` - это сообщение, которое необходимо декодировать.

## Использование

Приложение получает на вход валидное сообщение в виде классического кода Хэмминга любой длины и выдает ответ о том, что в сообщении нет ошибок, либо сообщение с указанием разряда ошибочного бита и далее – информационные биты исправленного сообщения.

## Скачивание

Собранные файлы jar можно загрузить из репозитория Github CI этого проекта. Файлы находятся в разделе "Actions". 
В случае, если необходимо собрать приложение самостоятельно, то следует скачать репозиторий и, находясь в директории проекта, запустить команду `mvn -B package --file pom.xml`.
